# Tags

Tags let you categorize and organize work items with color-coded labels. Common examples include "Electrical", "Plumbing", "Exterior", "Interior", or "Permits".

## Creating Tags

Navigate to **Tags** in the sidebar to access the tag management page. Click **New Tag** to create a tag with:

- **Name** -- A short label (e.g., "Plumbing")
- **Color** -- Pick from a color palette using the color picker

## Using Tags

When creating or editing a work item, select one or more tags from the tag dropdown. Tags appear as colored pills throughout the interface for quick visual identification.

## Managing Tags

From the tag management page you can:

- **Edit** a tag's name or color
- **Delete** a tag (this removes it from all work items that use it)
