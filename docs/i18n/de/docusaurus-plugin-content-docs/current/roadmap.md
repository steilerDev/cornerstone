# Roadmap

Cornerstone ist unter aktiver Entwicklung. Hier ist der aktuelle Stand geplanter Funktionen:

## Abgeschlossen

- [x] **EPIC-02: Anwendungs-Shell und Infrastruktur** ([#2](https://github.com/steilerDev/cornerstone/issues/2)) -- Responsive Layout, Routing, API-Client, Health Checks, Error Handling
- [x] **EPIC-11: CI/CD-Infrastruktur** ([#12](https://github.com/steilerDev/cornerstone/issues/12)) -- Automatisierte Builds, semantische Versionierung, Docker-Image Publishing
- [x] **EPIC-01: Authentifizierung und Benutzerverwaltung** ([#1](https://github.com/steilerDev/cornerstone/issues/1)) -- Lokale Anmeldung, OIDC SSO, Benutzerprofile, Admin-Panel, rollenbasierter Zugriff
- [x] **EPIC-03: Arbeitselemente** ([#3](https://github.com/steilerDev/cornerstone/issues/3)) -- Work Item CRUD, Tags, Notizen, Unteraufgaben, Abhängigkeiten, Tastenkombinationen, List und Detail Pages
- [x] **EPIC-12: Design System Bootstrap** ([#115](https://github.com/steilerDev/cornerstone/issues/115)) -- Design Token System, Dunkler Modus, Brand Identity, CSS Module Migration, Style Guide
- [x] **EPIC-05: Budget-Verwaltung** ([#5](https://github.com/steilerDev/cornerstone/issues/5)) -- Budget-Kategorien, Finanzierungsquellen, Work Item Kosten-Tracking, Lieferanten-Invoices, Zuschuss-Programme, Budget-Übersichts-Dashboard
- [x] **EPIC-06: Zeitplan und Gantt-Diagramm** ([#6](https://github.com/steilerDev/cornerstone/issues/6)) -- Interaktives Gantt-Diagramm, Kalenderansicht, Meilensteine, CPM-basierte Planungs-Engine, Abhängigkeitspfeile, kritische Pfad-Hervorhebung
- [x] **EPIC-08: Paperless-ngx-Integration** ([#8](https://github.com/steilerDev/cornerstone/issues/8)) -- Dokument-Browser, Dokument-Linking zu Work Items und Invoices, Proxy-Architektur, Thumbnail und Metadaten Display
- [x] **EPIC-04: Haushaltsgegenstände** ([#4](https://github.com/steilerDev/cornerstone/issues/4)) -- Möbel und Geräte-Tracking mit Kategorien, Lieferzeitplanung, Budget-Integration, Work Item Linking, Zeitplan-Abhängigkeiten, Invoice und Dokument-Linking
- [x] **EPIC-07: Reporting und Export** ([#7](https://github.com/steilerDev/cornerstone/issues/7)) -- Budget-Report-Generierung und Dokument-Export für Bankberichte
- [x] **EPIC-10: UX-Polish und Barrierefreiheit** ([#10](https://github.com/steilerDev/cornerstone/issues/10)) -- Cross-cutting UX-Verbesserungen, Tastenkombinationen, Touch-Optimierung, Performance-Tuning
- [x] **EPIC-11: Einheitliches Tag- und Kategorisierungs-System** ([#444](https://github.com/steilerDev/cornerstone/issues/444)) -- Konsolidiertes Tagging und Kategorisierung über Work Items, Budget Lines und Household Items
- [x] **EPIC-12: Codebasis-Refinement und Konsistenz** ([#445](https://github.com/steilerDev/cornerstone/issues/445)) -- Geteilte Route Factories, standardisierte API Clients, konsolidiertes Error Handling, wiederverwendbare Modal-Komponente
- [x] **EPIC-14: Cross-Entity Code Deduplication** ([#495](https://github.com/steilerDev/cornerstone/issues/495)) -- Geteilte Service Factories, einheitliche React Components für Budget und Subsidy Logic, Barrierefreiheits-Verbesserungen, harmonisierte Design Tokens
- [x] **EPIC-15: Budget-Line Invoice Linking Rework** ([#602](https://github.com/steilerDev/cornerstone/issues/602)) -- Many-to-Many Invoice-Budget-Line Linking mit itemisierten Beträgen, bidirektionales Linking UI, Invoice Groups auf Item Detail Pages, Subsidy Recalculation mit itemisierten Beträgen
- [x] **EPIC-09: Dashboard und Übersicht** ([#9](https://github.com/steilerDev/cornerstone/issues/9)) -- Projekt-Dashboard mit Budget-Zusammenfassung, Zeitplanstatus, Invoice und Zuschuss-Pipelines, Mini-Gantt-Vorschau und anpassbares Kartenlayout
- [x] **EPIC-13: Konstruktionstagebuch** ([#446](https://github.com/steilerDev/cornerstone/issues/446)) -- Konstruktionstagebuch mit manuellen Einträgen, automatische System-Events, Foto-Anhänge und Signaturfassung

## Geplant

Es gibt derzeit keine geplanten Epics. Verfolgen Sie den Fortschritt in Echtzeit auf dem [GitHub Projects Board](https://github.com/users/steilerDev/projects/4).
