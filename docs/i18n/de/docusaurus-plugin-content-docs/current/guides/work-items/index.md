# Arbeitselemente

Arbeitselemente sind das Herzstück von Cornerstone. Jedes Arbeitselement stellt eine Aufgabe in Ihrem Bauprojekt dar -- von "Küchenschränke installieren" bis "Elektrische Inspektionen einplanen".

## Übersicht

Arbeitselemente unterstützen:

- **Status** -- Nicht begonnen, In Bearbeitung, Abgeschlossen oder Blockiert
- **Datum und Planung** -- Startdatum, Enddatum, Dauer und Planungsbeschränkungen
- **Benutzerzuweisung** -- Aufgaben auf jeden Benutzer in Ihrer Instanz zuweisen
- **Tags** -- Farbkodierte Bezeichnungen für Kategorisierung (z.B. "Electrical", "Plumbing")
- **Notizen** -- Zeitgestempelte Kommentare mit Autorenzuordnung
- **Unteraufgaben** -- Checklisten-Elemente im Arbeitselement
- **Abhängigkeiten** -- Beziehungen zwischen Arbeitselementen (was vor oder nach vorkommen muss)
- **Dokumentlinks** -- Anhang von Dokumenten aus [Paperless-ngx](/guides/documents) (Verträge, Quittungen, Pläne)

## List View

Die Work Items List Page bietet:

- **Search** -- Volltextsuche mit verzögertem Input für schnelle Ergebnisse
- **Filtering** -- Filter nach Status, zugewiesenem Benutzer oder Tag
- **Sorting** -- Sortierung nach Titel, Status, Startdatum, Enddatum, Erstellungsdatum oder Aktualisierungsdatum
- **Pagination** -- Paginierte Ergebnisse für große Projekte
- **Responsive Layout** -- Tabellenansicht auf Desktop, Kartenansicht auf Mobile und Tablet

Alle Filter- und Sortiereinstellungen werden in der URL synchronisiert, sodass Ihre Ansicht hinzufügbar und teilbar ist.

![Work items list page](/img/screenshots/work-items-list-light.png)

## Detail View

Klicken Sie auf ein beliebiges Arbeitselement, um seine vollständige Detail-Seite mit allen Feldern, Notizen, Unteraufgaben, Abhängigkeiten und verknüpften Dokumenten zu sehen.
